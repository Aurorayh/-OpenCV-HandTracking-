# 双人贪吃蛇手势对战游戏（OpenCV+HandTracking）

本项目是一个基于OpenCV摄像头和cvzone手势识别的双人对战贪吃蛇游戏。  
支持换食物图片、蛇头图片、游戏过程换背景音乐和音效、死亡音效、自动配置记忆。可自定义体验，适合娱乐或教学演示。

## 功能亮点

- 支持多人手势控制（左右蛇，两个玩家）
- 食物图片、左右蛇头图片可在游戏运行时热切换
- 背景音乐、左右蛇吃食物音效、死亡音效支持运行中随时更换
- 所有图片和音效更换后自动保存记忆，下次开机自动应用上次设置
- 图片自动缩放适应，蛇头食物均为裁剪圆形并融合主色
- 键盘交互简便，玩法直观
- 积分计数，胜负判定

## 启动依赖

- Python 3.6+
- OpenCV (`opencv-python`)
- numpy
- Pillow
- pygame
- [cvzone](https://github.com/cvzone/cvzone)
- hand tracking模型（cvzone自带）

安装所需依赖示例：
```bash
pip install opencv-python numpy pillow pygame cvzone
```

## 资源准备

- 游戏主文件 `main.py`
- 背景音乐（如 `bgm.mp3`），左右吃音效（如 `eat_left.wav`、`eat_right.wav`），死亡音效（如 `dead.wav`），至少放一段。
- 默认食物/蛇头图片（如 `xin.png`），可用任意 PNG/JPG。
- 放所有资源与main.py在同一路径。

## 快捷键说明

| 键        | 功能描述                   |
|-----------|---------------------------|
| 空格      | 开始或重开一局             |
| Q/q       | 退出游戏                   |
| F         | 更换食物图片（输入路径）    |
| H         | 更换左蛇头图片（输入路径）  |
| J         | 更换右蛇头图片（输入路径）  |
| B         | 更换背景音乐（输入mp3等路径）|
| C         | 更换左蛇吃食物音效         |
| V         | 更换右蛇吃食物音效         |
| D         | 更换死亡音效               |
| + / -     | 增减蛇自碰撞敏感度         |

## 玩法说明

- 一人用左手、一人用右手，摄像头轨迹控制，手食指尖为蛇头引导方向。
- 穿越吃到食物、变长计分，撞到自己蛇身则GameOver。
- 游戏胜利自动判定分数高的为赢家，支持无限对战。

## 配置记忆

所有音效/图片换后都会写入 `snake_game_config.json`，下次开启自动恢复，不丢自定义设置。
